# Explainable-AI-lab-4

# Лабораторная работа №4: Иллюзия восприятия (Adversarial Attacks & XAI)

Исследование устойчивости нейронных сетей к состязательным атакам и анализ методов интерпретируемости (XAI).

## Описание работы

В данной лабораторной работе рассматриваются:

1. **Атака FGSM (Fast Gradient Sign Method)** - быстрый градиентный метод создания состязательных примеров
2. **Атака PGD (Projected Gradient Descent)** - итеративный метод, "золотой стандарт" атак
3. **XAI (Explainable AI)** - применение метода Saliency для анализа поведения модели

## Цели работы

- Продемонстрировать уязвимость нейронных сетей к состязательным атакам
- Сравнить эффективность FGSM и PGD атак
- Исследовать, как изменяются тепловые карты важности при атаке

## Результаты экспериментов

### FGSM атака (epsilon=0.05)
- Исходный класс: Egyptian cat (43.03%)
- Целевой класс: Toaster
- Результат: топ-1 класс стал "cowboy hat" (95.55%)
- Вероятность тостера выросла на 4.45%

### PGD атака (epsilon=0.08, alpha=0.01, iterations=30)
- Топ-1 класс после атаки: toaster
- Уверенность: 100.00%
- Вероятность Egyptian cat: 0.00%
